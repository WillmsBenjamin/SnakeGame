## Implemented Features:
  * Multiple fruit types (ex. poisonous causes the snake length to shrink)
  * Option of classic (one fruit type), or new game modes
  * Option of solo play and local multiplayer
  * Detailed menu
  * Pause function
  * Optional wall gaps
